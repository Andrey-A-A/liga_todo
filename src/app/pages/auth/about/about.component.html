Приложение "Список задач"

Это учебное приложение написанное при использовании фрейморка Angular. В приложении реализован следующий функционал:

  • Выводится список текущих задач. Для этого эмулируется запрос к серверу, на котором размещен json файл со списком.
  • По мере выполнения задач, их можно отметить, как выполненные
  • Реализована возможность удаления задачи из списка
  • Реализована возможность добавления задачи в список
  • Добавлена фильтрация/поиск по названию
  • Реализована навигация посредством роутинга
  • В приложении используется авторизация пользователя, которая даёт возможность просматривать список задач.
  • Страница со списком защищена посредством использования Guard
  • Также реализовано ограничение доступа к бэкенду приложения, если у пользователя нет соответствующих прав. Для этого используется отдельный структурный элемент Http Interseptor
